# Media library

- `images/` — photography, thumbnails, backgrounds, and stills
- `logos/` — brand marks, logo variants, and source sheets
- `godly/` — Godly Revolution project videos
- `video/` — showreels, case studies, and web video
- `audio/` — music, voice-over, and sound effects
- `fonts/` — locally hosted typefaces and licences
- `icons/` — SVG and raster interface icons

Use lowercase, descriptive, hyphen-separated filenames. Optimise production assets before publishing.
